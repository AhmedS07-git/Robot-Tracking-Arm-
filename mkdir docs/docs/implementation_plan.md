Implementation Plan

feature 1: move robot
The program moves the robot to a new positon. The robot class updates its coordinates and speed values. A stauts message is displayed to the user showing the movement.

Feature 2: Camera Trackig
The program opens the webcam using OpenCV. Camera frames are processed to detect a target location. the Sensor Class updates detected target coordinates.

Feature 3: Robotic Arm Control
The program changes the robotic arm angle. The robot class stores the new arm position and displays the updated status.

Feature 4: Display Robot Status
The program disp;ays the robot's current position, speed, arm angle, and tracking status. This is then updated to the console

Feature 5: Save and load data 
The RoborManger class loads saved robot information from a file at startup. Before exiting, the program saves the current robot data.
