# Robot-Tracking-Arm-

It's a C++ robot tracking sim programt that ses OpenCV camera input and object-oriented program.

Features
- moves the robot's position
- controls the angle of the arm
- display's the robot status
- load and save robot data

Building

cmake -S , -B build
cmake --build 

bash
./build/robot_system
